This calendar is for public MTB races events anywhere in the world. We list any event (for free!) providing that you follow these guidelines:
  • Please search for your event before entering it. The calendar has many events in it and we would like to keep duplicates to a minimum.
  • Enter only public events. If you don’t want other people to come, it’s probably best to keep them in your diary, not on the interwebs.
  • We need locations for all events and we use Google to geolocate (i.e. find the location on the globe). Google doesn’t know everything (yet), so sometimes you might have to try and a few names for your location before Google gets it right. When in doubt, try using a less particular name (say Placerville, CA instead of Salmon Trail, Placerville, CA) and add more details in the event’s descriptions.
  • If you have an event series (like a local Wednesday night race), please enter each individual race as a separate event, then tagged with the series name like "US MTB Cup". This allows users to add specific events to their personal calendars.
  • If you want to change an event, either add a comment (see the bottom event’s page) or contact the site admins via the contact page.
  • You can edit your events after you enter them by clicking on the pencil icon next to the event name once you are logged in.
  • Please try and keep the language clean.
  • The site admins reserve the right to edit or delete duplicate or inappropriate events from the calendar.
Enter a location so that we can add it to the map. We use a variety of geocoder (computer programs that plot location names to gps coordinates) to compute the location. Sometimes these programs cannot resolve your name with their databases. When that happens, you will see a red square around your location. You can either enter a new location or leave it to us to correct.
Choose a few meaningful and specific tags (separated by commas) to help collect your event with other similar events. Good tags include tags like the series name ("AMBC", "SERC", etc.). Poor tags include the obvious (i.e. biking, mtb, xc). Remember that the event will automatically be tagged with it's region, country and event types.
Choose the type (or format) of race that will be held at this event. Most MTB events will be cross-country or downhill, but we list a large number of formats to cover almost every kind of race offered. If your race format doesn't fit into any of these categories, choose "other."
To gain insurance, worldwide/nation/regional accreditation or , some events require a racing license from a governing body. In the United States, USA Cycling tends to be most popular. For top pros, a UCI license is necessary. Some regional races, such as many in Oregon, have an OBRA (Oregon Bike Racing Association) license.
